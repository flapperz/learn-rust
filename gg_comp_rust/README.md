# gg_comp_rust

Small learning repository created as a result of working through
the Google "Comprehensive Rust" guide: https://google.github.io/comprehensive-rust

What you'll find
- Tiny exercises and examples (e.g. `d1_ex_collatz_seq`, `d1_ex_fib`) under `src/bin`.
- A minimal `main` to show basic crate structure.

Run
- Use `cargo run --bin <name>` from the project root to execute a sample (for example `cargo run --bin d1_ex_fib`).

Notes
- I made this to practice Rust concepts from the guide; treat it as a personal playground, not a production crate.
